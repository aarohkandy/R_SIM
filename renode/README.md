# Renode

Phase 12 adds Renode `.resc` scripts, `.repl` platform files, and co-simulation glue for
the ESP32 and Teensy firmware.

Current files are intentionally marked as placeholders until real firmware ELFs and
verified board/peripheral models are supplied. Run `make hil` to generate the current
bring-up status under `outputs/phase12_renode_hil_status/`.

Renode is not required for early phases. Installation notes live in `docs/tooling.md`.
